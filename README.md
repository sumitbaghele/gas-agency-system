# Gas Agency System

## Technologies Used

* HTML
* CSS
* JavaScript
* Firebase

## Features

* User & Admin Login
* Gas Cylinder Booking
* Booking History
* Admin Approval System
* Email Notifications

## How to Run

1. Clone the repository
2. Open index.html in browser
3. Configure Firebase keys
4. Login as User or Admin

## Deployment

Project can be deployed using Firebase Hosting.
